### Mermaids 2.0.0 Changelog:

- Created a brand-new mermaid tail model!
  - The tail model currently supports all swim animations.
  - Currently, has multiple tail colors.
  - Retired the old tail model and assets.
- Your stamina depletes slower while swimming.
- Removed the /mermaids bugfix command.
- Renamed the /mermaids admin itemincreasespeed to itemspeed.
- Renamed the Mermaid model ID to Mermaids_Mermaid.
- Renamed all the potion ID's to began with Mermaids.
- Add compatibility with [Keyblades Reimagined](https://www.curseforge.com/hytale/mods/keyblades-reimagined) by [TaHie](https://www.curseforge.com/members/tahie/projects).
- Mermaid model will now show hands in first person.
- Added the Mermaids logo to the mod that will show in the world settings menu in the mods tab.
- Renamed the language file from items to server and added all commands to it.
- Language file changes.
  - Renamed Mermaid potion IDs.
  - Changed the language file path to all items to include item.mermaids.ITEMPATH.
  - Renamed all mermaid ui language file path.
  - Added a description to a few custom items that has been in the Mermaids mod that has been added.
  - Changed the path for all customui.
- Removed the /mermaids debug mermaidsv2 command.
- Check for new version update every 6 hours.
- Mermaids mod now supports Hytale Update 4.
